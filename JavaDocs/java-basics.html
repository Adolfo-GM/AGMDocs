Java Basics

Let's learn the fundamental concepts of Java programming. Understanding these basics is essential for building more complex applications.

1. Java Program Structure

Every Java program has a specific structure. Here's a simple example:

// This is a comment public class MyProgram { // Main method - entry point of the program public static void main(String[] args) { System.out.println("Welcome to Java programming!"); } }

Key Points:
  • Every Java program must have at least one class
  • The class name must match the filename
  • The main method is where program execution begins
  • Java is case-sensitive

2. Comments

Comments help explain your code and are ignored by the compiler:

public class Comments { public static void main(String[] args) { // Single-line comment System.out.println("This line will print"); /* Multi-line comment can span multiple lines like this */ System.out.println("Comments are helpful!"); } }

3. Variables and Data Types

Java has several built-in data types to store different kinds of information:

Primitive Data Types

public class DataTypes { public static void main(String[] args) { // Integer types int age = 25; long population = 7800000000L; // Floating point types double price = 19.99; float temperature = 98.6f; // Character and boolean char grade = 'A'; boolean isStudent = true; // String (not primitive, but commonly used) String name = "John Doe"; // Print all variables System.out.println("Name: " + name); System.out.println("Age: " + age); System.out.println("Grade: " + grade); System.out.println("Is Student: " + isStudent); System.out.println("Price: $" + price); } }

4. Basic Input and Output

Learn how to get input from users and display output:

import java.util.Scanner; public class InputOutput { public static void main(String[] args) { // Create a Scanner object for input Scanner scanner = new Scanner(System.in); // Get user input (simulated for this example) System.out.print("Enter your name: "); String userName = "Alice"; // scanner.nextLine(); System.out.print("Enter your age: "); int userAge = 22; // scanner.nextInt(); // Display output System.out.println("Hello, " + userName + "!"); System.out.println("You are " + userAge + " years old."); // Close the scanner scanner.close(); } }

5. String Operations

Strings are objects in Java with many useful methods:

public class StringOperations { public static void main(String[] args) { String message = "Hello, Java World!"; // String properties System.out.println("Original: " + message); System.out.println("Length: " + message.length()); // String methods System.out.println("Uppercase: " + message.toUpperCase()); System.out.println("Lowercase: " + message.toLowerCase()); System.out.println("Contains 'Java': " + message.contains("Java")); System.out.println("Substring: " + message.substring(7, 11)); // String concatenation String greeting = "Welcome"; String fullGreeting = greeting + " to Java programming!"; System.out.println(fullGreeting); } }

6. Mathematical Operations

Java provides standard mathematical operators and the Math class:

public class MathOperations { public static void main(String[] args) { int a = 10; int b = 3; // Basic arithmetic System.out.println("Addition: " + a + " + " + b + " = " + (a + b)); System.out.println("Subtraction: " + a + " - " + b + " = " + (a - b)); System.out.println("Multiplication: " + a + " * " + b + " = " + (a * b)); System.out.println("Division: " + a + " / " + b + " = " + (a / b)); System.out.println("Remainder: " + a + " % " + b + " = " + (a % b)); // Math class methods double num = 16.5; System.out.println("Square root of " + num + ": " + Math.sqrt(num)); System.out.println("Power of 2^3: " + Math.pow(2, 3)); System.out.println("Maximum of " + a + " and " + b + ": " + Math.max(a, b)); } }
