Spring Framework

The Spring Framework is a popular application framework and inversion of control container for the Java platform.

1. Basic Spring Boot Application

Here's a simple "Hello, World!" example using Spring Boot.

import org.springframework.boot.SpringApplication; import org.springframework.boot.autoconfigure.SpringBootApplication; import org.springframework.web.bind.annotation.GetMapping; import org.springframework.web.bind.annotation.RestController; @SpringBootApplication @RestController public class DemoApplication { public static void main(String[] args) { SpringApplication.run(DemoApplication.class, args); } @GetMapping("/") public String hello() { return "Hello, World!"; } }
