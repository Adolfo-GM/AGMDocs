Control Flow in Java

Control flow statements allow you to control the execution order of your program. Java provides several types of control structures including conditional statements and loops.

1. If-Else Statements

Conditional statements execute different code blocks based on boolean conditions:

public class IfElseExample { public static void main(String[] args) { // Simple if statement int age = 18; if (age >= 18) { System.out.println("You are an adult."); } // If-else statement int temperature = 25; if (temperature > 30) { System.out.println("It's hot today!"); } else { System.out.println("It's a nice day!"); } // If-else if-else chain int score = 85; if (score >= 90) { System.out.println("Grade: A"); } else if (score >= 80) { System.out.println("Grade: B"); } else if (score >= 70) { System.out.println("Grade: C"); } else if (score >= 60) { System.out.println("Grade: D"); } else { System.out.println("Grade: F"); } // Nested if statements boolean hasLicense = true; int driverAge = 20; if (driverAge >= 16) { if (hasLicense) { System.out.println("You can drive!"); } else { System.out.println("You need a license to drive."); } } else { System.out.println("You're too young to drive."); } // Complex conditions with logical operators int hour = 14; boolean isWeekend = false; if (hour >= 9 && hour <= 17 && !isWeekend) { System.out.println("Office hours - Time to work!"); } else if (isWeekend || hour < 9 || hour > 17) { System.out.println("Free time - Enjoy yourself!"); } } }

2. Switch Statements

Switch statements provide a cleaner way to handle multiple conditions based on a single variable:

public class SwitchExample { public static void main(String[] args) { // Basic switch statement with int int dayOfWeek = 3; switch (dayOfWeek) { case 1: System.out.println("Monday - Start of work week"); break; case 2: System.out.println("Tuesday - Getting into the groove"); break; case 3: System.out.println("Wednesday - Hump day!"); break; case 4: System.out.println("Thursday - Almost there"); break; case 5: System.out.println("Friday - TGIF!"); break; case 6: case 7: System.out.println("Weekend - Time to relax!"); break; default: System.out.println("Invalid day"); } // Switch with String (Java 7+) String grade = "B"; switch (grade) { case "A": System.out.println("Excellent! Score: 90-100"); break; case "B": System.out.println("Good job! Score: 80-89"); break; case "C": System.out.println("Average. Score: 70-79"); break; case "D": System.out.println("Below average. Score: 60-69"); break; case "F": System.out.println("Failed. Score: Below 60"); break; default: System.out.println("Invalid grade"); } // Switch with char char operation = '+'; double a = 10, b = 5, result = 0; switch (operation) { case '+': result = a + b; System.out.println(a + " + " + b + " = " + result); break; case '-': result = a - b; System.out.println(a + " - " + b + " = " + result); break; case '*': result = a * b; System.out.println(a + " * " + b + " = " + result); break; case '/': if (b != 0) { result = a / b; System.out.println(a + " / " + b + " = " + result); } else { System.out.println("Cannot divide by zero!"); } break; default: System.out.println("Invalid operation"); } } }

3. For Loops

For loops are perfect for iterating a specific number of times or through collections:

public class ForLoopsExample { public static void main(String[] args) { // Basic for loop System.out.println("=== Basic For Loop ==="); for (int i = 1; i <= 5; i++) { System.out.println("Count: " + i); } // For loop with different increments System.out.println("\n=== Even Numbers ==="); for (int i = 2; i <= 10; i += 2) { System.out.println("Even: " + i); } // For loop counting backwards System.out.println("\n=== Countdown ==="); for (int i = 5; i >= 1; i--) { System.out.println("Countdown: " + i); } System.out.println("Blast off!"); // For loop with arrays System.out.println("\n=== Array Iteration ==="); int[] numbers = {10, 20, 30, 40, 50}; for (int i = 0; i < numbers.length; i++) { System.out.println("Index " + i + ": " + numbers[i]); } // Enhanced for loop (for-each) System.out.println("\n=== Enhanced For Loop ==="); String[] fruits = {"Apple", "Banana", "Orange", "Grape"}; for (String fruit : fruits) { System.out.println("Fruit: " + fruit); } // Nested for loops System.out.println("\n=== Multiplication Table ==="); for (int i = 1; i <= 3; i++) { for (int j = 1; j <= 3; j++) { System.out.print(i + " x " + j + " = " + (i * j) + " "); } System.out.println(); // New line after each row } } }

4. While and Do-While Loops

While loops continue executing as long as a condition remains true:

public class WhileLoopsExample { public static void main(String[] args) { // Basic while loop System.out.println("=== Basic While Loop ==="); int count = 1; while (count <= 5) { System.out.println("Count: " + count); count++; } // While loop with condition System.out.println("\n=== Number Doubling ==="); int number = 1; while (number < 100) { System.out.println("Number: " + number); number *= 2; } // Do-while loop (executes at least once) System.out.println("\n=== Do-While Loop ==="); int value = 10; do { System.out.println("Value: " + value); value--; } while (value > 7); // Do-while example that runs even when condition is false System.out.println("\n=== Do-While (Condition Initially False) ==="); int x = 15; do { System.out.println("This will execute once, x = " + x); } while (x < 10); // False from the start // While loop for input validation simulation System.out.println("\n=== Simulated Input Validation ==="); int attempts = 0; int maxAttempts = 3; boolean success = false; while (attempts < maxAttempts && !success) { attempts++; System.out.println("Attempt " + attempts + ": Trying to connect..."); // Simulate success on 2nd attempt if (attempts == 2) { success = true; System.out.println("Connection successful!"); } else { System.out.println("Connection failed. Retrying..."); } } if (!success) { System.out.println("Max attempts reached. Connection failed."); } } }

5. Break and Continue

Break and continue statements control loop execution flow:

public class BreakContinueExample { public static void main(String[] args) { // Break statement - exits the loop System.out.println("=== Break Example ==="); for (int i = 1; i <= 10; i++) { if (i == 6) { System.out.println("Breaking at " + i); break; // Exit the loop when i equals 6 } System.out.println("Number: " + i); } // Continue statement - skips current iteration System.out.println("\n=== Continue Example ==="); for (int i = 1; i <= 10; i++) { if (i % 2 == 0) { continue; // Skip even numbers } System.out.println("Odd number: " + i); } // Break in while loop System.out.println("\n=== Break in While Loop ==="); int count = 0; while (true) { // Infinite loop count++; if (count > 5) { System.out.println("Reached limit, breaking..."); break; } System.out.println("Count: " + count); } // Break and continue in nested loops System.out.println("\n=== Nested Loops with Break/Continue ==="); for (int i = 1; i <= 3; i++) { System.out.println("Outer loop: " + i); for (int j = 1; j <= 5; j++) { if (j == 3) { System.out.println(" Skipping j = " + j); continue; // Skip j = 3 } if (j == 4) { System.out.println(" Breaking inner loop at j = " + j); break; // Exit inner loop when j = 4 } System.out.println(" Inner loop: " + j); } } // Practical example: Finding first even number in array System.out.println("\n=== Finding First Even Number ==="); int[] numbers = {1, 3, 7, 8, 9, 12, 15}; for (int num : numbers) { System.out.println("Checking: " + num); if (num % 2 == 0) { System.out.println("Found first even number: " + num); break; } } } }

Control Flow Summary:
  • if-else: Execute code based on conditions
  • switch: Multiple conditions based on one variable
  • for loop: Iterate a specific number of times
  • while loop: Continue while condition is true
  • do-while: Execute at least once, then check condition
  • break: Exit loop immediately
  • continue: Skip current iteration, continue with next

6. Best Practices

  • Use meaningful variable names in loop conditions
  • Avoid infinite loops by ensuring loop conditions will eventually become false
  • Use enhanced for loops when you don't need the index
  • Consider using switch statements for multiple conditions on the same variable
  • Always include break statements in switch cases (unless fall-through is intended)
  • Use continue sparingly - sometimes restructuring with if statements is clearer
  • Keep nested loops to a minimum for better readability
