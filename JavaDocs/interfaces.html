Java Interfaces

An interface in Java is a blueprint of a class. It has static constants and abstract methods.

1. Defining and Implementing an Interface

Here's how to define an interface and implement it in a class.

// Interface interface Drawable { void draw(); } // Implementation class Circle implements Drawable { public void draw() { System.out.println("Drawing a circle"); } } public class TestInterface { public static void main(String[] args) { Drawable d = new Circle(); d.draw(); } }
