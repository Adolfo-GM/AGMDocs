Python Math Module

Python's built-in math module provides access to common mathematical functions and constants. It's useful for more advanced numerical operations beyond basic arithmetic.

Importing the Math Module

Before you can use functions from the math module, you need to import it:

import math print("Math module imported successfully!")

Basic Mathematical Functions

Functions for common mathematical operations.

import math # Absolute value num = -10.5 print(f"Absolute value of {num}: {math.fabs(num)}") # Square root x = 25 print(f"Square root of {x}: {math.sqrt(x)}") # Power base = 2 exponent = 3 print(f"{base} raised to the power of {exponent}: {math.pow(base, exponent)}") # Ceiling and Floor val = 4.7 print(f"Ceiling of {val}: {math.ceil(val)}") print(f"Floor of {val}: {math.floor(val)}")

Logarithmic and Trigonometric Functions

The math module also provides functions for logarithms and trigonometry.

import math # Logarithms num = 100 print(f"Natural logarithm of {num}: {math.log(num)}") print(f"Logarithm base 10 of {num}: {math.log10(num)}") # Trigonometric functions (angles in radians) angle_degrees = 90 angle_radians = math.radians(angle_degrees) print(f"Sine of {angle_degrees} degrees: {math.sin(angle_radians)}") print(f"Cosine of {angle_degrees} degrees: {math.cos(angle_radians)}") print(f"Tangent of {angle_degrees} degrees: {math.tan(angle_radians)}")

Important Notes
  • Most trigonometric functions in the math module expect angles in radians. Use math.radians() to convert degrees to radians.
  • The math module does not handle complex numbers. For complex number operations, use the cmath module.
