Python Variables

Variables are containers for storing data values. Python has several built-in data types that you can use to store different kinds of information.

What is a Variable?

A variable is like a labeled box where you can store information. You can put different types of data in these boxes and refer to them by their name (label) whenever you need the data.

Creating Variables

In Python, you create variables by simply assigning values to them using the = operator. No need to declare the type beforehand!

# Creating different variables name = "Alice" age = 25 height = 5.6 is_student = True print("Name:", name) print("Age:", age) print("Height:", height) print("Is student:", is_student)

Python Data Types

Python has several built-in data types. Here's a comprehensive overview:

Data Type Description Example
int Whole numbers (positive or negative) 42, -17, 0
float Decimal numbers 3.14, -2.5, 0.0
str Text (strings) "Hello", 'Python'
bool True or False values True, False
list Ordered collection of items [1, 2, 3], ["a", "b"]
tuple Immutable ordered collection (1, 2, 3), ("x", "y")
dict Key-value pairs {"name": "Bob", "age": 30}
set Unordered collection of unique items {1, 2, 3}, {"a", "b"}

Integers are whole numbers without decimal points. They can be positive, negative, or zero.

Floating Point Numbers (float)

Floats are numbers that contain decimal points. They're used for precise calculations.

Strings (str)

Strings are sequences of characters used to store text. They can be enclosed in single or double quotes.

Booleans (bool)

Booleans represent truth values: either True or False. They're essential for logical operations and conditions.

# Boolean examples is_sunny = True is_raining = False has_umbrella = True print("Is sunny:", is_sunny) print("Is raining:", is_raining) print("Has umbrella:", has_umbrella) # Boolean operations need_jacket = not is_sunny go_outside = is_sunny or has_umbrella print("Need jacket:", need_jacket) print("Go outside:", go_outside) print("Type:", type(is_sunny))

Lists

Lists are ordered collections that can store multiple items. They're mutable (can be changed) and allow duplicates.

# List examples numbers = [1, 2, 3, 4, 5] fruits = ["apple", "banana", "orange"] mixed = [1, "hello", 3.14, True] empty_list = [] print("Numbers:", numbers) print("Fruits:", fruits) print("Mixed:", mixed) print("First fruit:", fruits[0]) print("List length:", len(fruits)) # Adding items fruits.append("grape") print("After adding grape:", fruits) print("Type:", type(numbers))

Tuples

Tuples are similar to lists but are immutable (cannot be changed after creation). They're perfect for storing related data that shouldn't change.

# Tuple examples coordinates = (10, 20) rgb_color = (255, 128, 0) person = ("Alice", 25, "Engineer") single_item = (42,) # Note the comma! print("Coordinates:", coordinates) print("RGB Color:", rgb_color) print("Person:", person) print("X coordinate:", coordinates[0]) print("Y coordinate:", coordinates[1]) # Unpacking tuples name, age, job = person print(f"Name: {name}, Age: {age}, Job: {job}") print("Type:", type(coordinates))

Dictionaries

Dictionaries store data in key-value pairs. They're perfect for storing related information that you want to access by name rather than position.

# Dictionary examples student = { "name": "Bob", "age": 20, "grade": "A", "subjects": ["Math", "Science", "English"] } empty_dict = {} scores = {"math": 95, "science": 87, "english": 92} print("Student:", student) print("Student name:", student["name"]) print("Student age:", student["age"]) print("Subjects:", student["subjects"]) # Adding new key-value pair student["city"] = "New York" print("Updated student:", student) print("Type:", type(student))

Sets

Sets are unordered collections of unique items. They automatically remove duplicates and are great for membership testing.

# Set examples unique_numbers = {1, 2, 3, 3, 4, 4, 5} colors = {"red", "green", "blue"} empty_set = set() # {} creates an empty dict, not set! print("Unique numbers:", unique_numbers) # Duplicates removed print("Colors:", colors) print("Set length:", len(colors)) # Set operations colors.add("yellow") print("After adding yellow:", colors) # Check membership print("Is red in colors?", "red" in colors) print("Is purple in colors?", "purple" in colors) print("Type:", type(colors))

Type Checking and Conversion

Python provides functions to check and convert between different data types.

# Type checking x = 42 y = "Hello" z = 3.14 print(f"x = {x}, type: {type(x)}") print(f"y = {y}, type: {type(y)}") print(f"z = {z}, type: {type(z)}") # Type conversion num_str = "123" num_int = int(num_str) num_float = float(num_str) print(f"Original: '{num_str}' (type: {type(num_str)})") print(f"As int: {num_int} (type: {type(num_int)})") print(f"As float: {num_float} (type: {type(num_float)})") # Converting back to string back_to_str = str(num_int) print(f"Back to string: '{back_to_str}' (type: {type(back_to_str)})")

Important Notes
  • Variable names can contain letters, numbers, and underscores but cannot start with a number
  • Python is case-sensitive: name and Name are different variables
  • Use descriptive names: student_age is better than a
  • Constants are usually written in UPPERCASE: MAX_SIZE = 100
