Python Random Module

The random module in Python is used to generate pseudo-random numbers. It's useful for simulations, games, security applications (though for cryptographic security, use secrets module), and more.

Importing the Random Module

Before you can use functions from the random module, you need to import it:

import random print("Random module imported successfully!")

Generating Random Integers: randint()

The randint(a, b) function returns a random integer N such that a <= N <= b.

import random # Generate a random integer between 1 and 10 (inclusive) random_number = random.randint(1, 10) print(f"Random number between 1 and 10: {random_number}") # Simulate a dice roll dice_roll = random.randint(1, 6) print(f"Dice roll: {dice_roll}")

Generating Random Floats: random() and uniform()

random() returns a random float in the range [0.0, 1.0). uniform(a, b) returns a random float N such that a <= N <= b.

import random # Random float between 0.0 and 1.0 rand_float_0_1 = random.random() print(f"Random float (0.0 to 1.0): {rand_float_0_1}") # Random float between 10.0 and 20.0 rand_float_10_20 = random.uniform(10.0, 20.0) print(f"Random float (10.0 to 20.0): {rand_float_10_20}")

Choosing from a Sequence: choice() and choices()

choice(sequence) returns a random element from a non-empty sequence. choices(sequence, k=n) returns a list of n random elements with replacement.

import random # Choose a random fruit fruits = ["apple", "banana", "cherry", "date"] selected_fruit = random.choice(fruits) print(f"Random fruit: {selected_fruit}") # Choose multiple random items with replacement lucky_numbers = random.choices(range(1, 100), k=5) print(f"Five lucky numbers: {lucky_numbers}")

Shuffling a Sequence: shuffle()

The shuffle(x) function shuffles the items of a list in place (modifies the original list).

import random # Shuffle a deck of cards (represented as a list) deck = ['Ace', 'King', 'Queen', 'Jack', '10'] print(f"Original deck: {deck}") random.shuffle(deck) print(f"Shuffled deck: {deck}")

Important Notes
  • The random module uses a pseudo-random number generator, meaning the numbers are not truly random but are generated by an algorithm.
  • For cryptographic or security-sensitive applications, use the secrets module, which provides a more secure way to generate random numbers.
  • To reproduce a sequence of random numbers, you can use random.seed() with a specific value.
