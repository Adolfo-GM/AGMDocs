Get Started

Welcome to the Python setup guide! Follow these step-by-step instructions to install Python and VS Code on your computer.

Step 1: Install Python

First, we need to install Python on your computer. Python is the programming language we'll be using.

For Windows:

  1. Go to the official Python website: python.org
  2. Click on the "Download Python" button (it will show the latest version)
  3. Run the downloaded installer (.exe file)
  4. Important: Check the box "Add Python to PATH" during installation
  5. Click "Install Now"
  6. Wait for the installation to complete

For macOS:

  1. Go to the official Python website: python.org
  2. Click on the "Download Python" button
  3. Open the downloaded .pkg file
  4. Follow the installation wizard
  5. Enter your password when prompted

Alternative: If you have Homebrew installed, you can run:

brew install python

For Linux (Ubuntu/Debian):

  1. Open Terminal
  2. Update package list:
sudo apt update

sudo apt install python3 python3-pip

Step 2: Verify Python Installation

Let's make sure Python was installed correctly:

  1. Open Terminal (macOS/Linux) or Command Prompt (Windows)
  2. Type the following command and press Enter:
python --version
Terminal Output
Click "Run" to see output...

You should see something like "Python 3.11.x" or similar. If you get an error, try:

python3 --version
Terminal Output
Click "Run" to see output...
Step 3: Install Visual Studio Code

VS Code is a powerful code editor that makes writing Python code much easier.

  1. Go to the VS Code website: code.visualstudio.com
  2. Click "Download for [Your Operating System]"
  3. Run the installer and follow the setup wizard
  4. Launch VS Code when installation is complete
Step 4: Install Python Extension for VS Code

To make Python development easier in VS Code, we need to install the Python extension.

  1. Open VS Code
  2. Click on the Extensions icon in the left sidebar (or press Ctrl+Shift+X)
  3. Search for "Python"
  4. Install the extension by Microsoft (it should be the first result)
  5. Restart VS Code after installation
Step 5: Test Your Setup

Let's create your first Python program to make sure everything works!

  1. Open VS Code
  2. Create a new file (File → New File)
  3. Save it as "hello.py" (File → Save As)
  4. Type the following code:
print("Hello, World!") print("Python is working!")
Terminal Output

  1. Save the file (Ctrl+S or Cmd+S)
  2. Right-click in the editor and select "Run Python File in Terminal"
  3. You should see the output in the terminal at the bottom of VS Code
Troubleshooting

If you encounter any issues:

  • Make sure Python is added to your system PATH
  • Try restarting your computer after installation
  • On Windows, you might need to use "py" instead of "python" command
  • Make sure you have the latest version of VS Code
