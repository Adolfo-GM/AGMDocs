Pygame Introduction

Pygame is a popular Python library for building 2D games and multimedia applications. It provides modules for graphics, sound, input, and more, built on top of SDL.

Installation Required

Before using Pygame, install it with: pip install pygame
Use a virtual environment for cleaner dependency management.

Basic Game Window

Every Pygame program follows an init, loop, quit pattern. The loop processes events and updates the screen at a controlled frame rate.

# Basic Pygame window import pygame, sys pygame.init() WIDTH, HEIGHT = 640, 360 screen = pygame.display.set_mode((WIDTH, HEIGHT)) pygame.display.set_caption("Pygame Window") clock = pygame.time.Clock() # Main loop while True: for event in pygame.event.get(): if event.type == pygame.QUIT: pygame.quit() sys.exit() screen.fill((30, 30, 40)) pygame.display.flip() clock.tick(60) # limit to 60 FPS

Drawing Shapes & Text

Use the draw module and font rendering for HUD elements.

# Drawing shapes and text import pygame, sys pygame.init() screen = pygame.display.set_mode((500, 300)) font = pygame.font.SysFont("Arial", 24) clock = pygame.time.Clock() angle = 0 while True: for e in pygame.event.get(): if e.type == pygame.QUIT: pygame.quit() sys.exit() angle = (angle + 2) % 360 screen.fill((18, 18, 24)) # Draw shapes pygame.draw.rect(screen, (70, 130, 180), (40, 40, 120, 80), border_radius=12) pygame.draw.circle(screen, (220, 90, 90), (300, 150), 50) # Draw text txt = font.render(f"Angle: {angle}", True, (200, 200, 210)) screen.blit(txt, (20, 10)) pygame.display.flip() clock.tick(60)

Moving a Player (Keyboard Input)

Respond to key presses by adjusting a rect's position. Use pygame.key.get_pressed() for smooth movement.

# Movement with keyboard import pygame, sys pygame.init() screen = pygame.display.set_mode((520, 320)) clock = pygame.time.Clock() player = pygame.Rect(240, 140, 40, 40) speed = 4 while True: for e in pygame.event.get(): if e.type == pygame.QUIT: pygame.quit() sys.exit() # Handle movement keys = pygame.key.get_pressed() player.x += (keys[pygame.K_RIGHT] - keys[pygame.K_LEFT]) * speed player.y += (keys[pygame.K_DOWN] - keys[pygame.K_UP]) * speed # Draw everything screen.fill((25, 25, 32)) pygame.draw.rect(screen, (255, 180, 55), player, border_radius=6) pygame.display.flip() clock.tick(60)

Basic Collision Detection

Use pygame.Rect intersection methods for collision detection.

# Rect collision detection import pygame, sys, random pygame.init() screen = pygame.display.set_mode((520, 320)) clock = pygame.time.Clock() player = pygame.Rect(40, 40, 40, 40) food = pygame.Rect(300, 180, 26, 26) score = 0 font = pygame.font.SysFont("Arial", 20) speed = 5 while True: for e in pygame.event.get(): if e.type == pygame.QUIT: pygame.quit() sys.exit() # Movement with WASD keys k = pygame.key.get_pressed() player.x += (k[pygame.K_d] - k[pygame.K_a]) * speed player.y += (k[pygame.K_s] - k[pygame.K_w]) * speed # Collision detection if player.colliderect(food): score += 1 food.x = random.randrange(0, 520 - 26) food.y = random.randrange(0, 320 - 26) # Drawing screen.fill((20, 22, 28)) pygame.draw.rect(screen, (50, 200, 90), player, border_radius=4) pygame.draw.rect(screen, (230, 60, 80), food) score_text = font.render(f"Score: {score}", True, (220, 220, 230)) screen.blit(score_text, (10, 8)) pygame.display.flip() clock.tick(60)

Suggested Project Structure
  • main.py: Entry point; creates window & runs loop.
  • settings.py: Constants (WIDTH, HEIGHT, FPS).
  • player.py: Player sprite class.
  • scenes/: Menu, game, pause states.
  • assets/: Images, sounds, fonts.
