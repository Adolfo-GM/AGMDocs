Tkinter Module

Tkinter is Python's de-facto standard GUI (Graphical User Interface) package. It is a standard Python interface to the Tcl/Tk GUI toolkit. Tkinter is the fastest and easiest way to create GUI applications.

Basic Tkinter Window

Here's a simple example of creating a basic Tkinter window:

import tkinter as tk # Create the main window root = tk.Tk() root.title("My First Tkinter Window") root.geometry("300x200") # Create a label widget label = tk.Label(root, text="Hello from Tkinter!") label.pack(pady=20) # Run the Tkinter event loop root.mainloop()

Widgets

Tkinter provides various widgets to build your GUI:

  • Label: Used to display text or images.
  • Button: Used to create clickable buttons.
  • Entry: Used for single-line text input.
  • Text: Used for multi-line text input.
  • Frame: A rectangular region used to organize other widgets.
  • Canvas: Used for drawing shapes and images.
  • Checkbutton: Creates checkboxes for toggling options.
  • Radiobutton: Creates radio buttons for selecting one option from many.
  • Listbox: Displays a list of options.
  • Scrollbar: Adds scrolling functionality to widgets.

Interactive Button Example

Here's an example of a Tkinter application with a button that responds to clicks:

import tkinter as tk # Function to handle button click def button_clicked(): label.config(text="Button was clicked!") # Change button color when clicked button.config(bg="lightgreen", activebackground="green") # Create the main window root = tk.Tk() root.title("Button Example") root.geometry("300x200") # Create a label widget label = tk.Label(root, text="Click the button below", font=("Arial", 12)) label.pack(pady=20) # Create a button widget button = tk.Button(root, text="Click Me!", command=button_clicked, bg="#0078D7", fg="white", activebackground="#00559B", activeforeground="white", padx=20, pady=10) button.pack() # Run the Tkinter event loop root.mainloop()

Form Example with Multiple Widgets

This example shows how to create a form with multiple widgets including entry fields, labels, and buttons:

import tkinter as tk from tkinter import messagebox def submit_form(): name = name_entry.get() email = email_entry.get() if name and email: result_label.config(text=f"Thank you, {name}!\nForm submitted successfully!", fg="green") else: messagebox.showerror("Error", "Please fill in all fields!") # Create the main window root = tk.Tk() root.title("Contact Form") root.geometry("400x300") # Create a frame for the form form_frame = tk.Frame(root, padx=20, pady=20) form_frame.pack(fill="both", expand=True) # Form title title_label = tk.Label(form_frame, text="Contact Information", font=("Arial", 14, "bold")) title_label.grid(row=0, column=0, columnspan=2, pady=10, sticky="w") # Name field name_label = tk.Label(form_frame, text="Name:") name_label.grid(row=1, column=0, sticky="w", pady=5) name_entry = tk.Entry(form_frame, width=30) name_entry.grid(row=1, column=1, sticky="w", pady=5) # Email field email_label = tk.Label(form_frame, text="Email:") email_label.grid(row=2, column=0, sticky="w", pady=5) email_entry = tk.Entry(form_frame, width=30) email_entry.grid(row=2, column=1, sticky="w", pady=5) # Submit button submit_button = tk.Button(form_frame, text="Submit", command=submit_form, bg="#0078D7", fg="white") submit_button.grid(row=3, column=1, sticky="w", pady=15) # Result label result_label = tk.Label(form_frame, text="") result_label.grid(row=4, column=0, columnspan=2, pady=5) root.mainloop()
