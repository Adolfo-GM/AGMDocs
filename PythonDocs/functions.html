Python Functions

Functions are reusable blocks of code that perform specific tasks. They help organize your code, make it more readable, and avoid repetition.

What is a Function?

A function is like a recipe. You give it some ingredients (parameters), it follows a set of instructions (the function body), and it gives you back a result (return value). Once you write a function, you can use it as many times as you want!

Basic Function Definition

To create a function in Python, use the def keyword followed by the function name and parentheses.

# Define a simple function def say_hello(): print("Hello, World!") print("Welcome to Python functions!") # Call the function say_hello() # You can call it multiple times say_hello() say_hello()

Functions with Parameters

Parameters allow you to pass data into functions, making them more flexible and reusable.

# Function with one parameter def greet_person(name): print(f"Hello, {name}!") print(f"Nice to meet you, {name}.") # Function with multiple parameters def introduce_person(name, age, city): print(f"Name: {name}") print(f"Age: {age}") print(f"City: {city}") # Call the functions greet_person("Alice") greet_person("Bob") introduce_person("Emma", 25, "New York")

Functions that Return Values

Functions can return values using the return statement. This allows you to get results back from the function.

# Function that returns a value def add_numbers(a, b): result = a + b return result def calculate_area(length, width): return length * width def get_full_name(first, last): return first + " " + last # Use the returned values sum_result = add_numbers(10, 5) print(f"10 + 5 = {sum_result}") area = calculate_area(8, 6) print(f"Area: {area} square units") name = get_full_name("John", "Smith") print(f"Full name: {name}")

Default Parameters

You can provide default values for parameters. If no argument is passed, the default value is used.

# Function with default parameters def greet(name, greeting="Hello"): return f"{greeting}, {name}!" def create_profile(name, age=0, city="Unknown"): return f"{name}, {age} years old, from {city}" # Call with different combinations print(greet("Alice")) # Uses default greeting print(greet("Bob", "Hi")) # Custom greeting print(greet("Charlie", "Good morning")) print(create_profile("Emma")) # Uses defaults print(create_profile("David", 30)) # Custom age print(create_profile("Sarah", 25, "Boston")) # All custom

Functions with Lists and Dictionaries

Functions can work with complex data types like lists and dictionaries.

# Function that works with lists def calculate_average(numbers): if len(numbers) == 0: return 0 return sum(numbers) / len(numbers) def find_largest(numbers): return max(numbers) # Function that works with dictionaries def display_student(student): print(f"Student: {student['name']}") print(f"Age: {student['age']}") print(f"Grade: {student['grade']}") # Test the functions scores = [85, 92, 78, 96, 89] avg = calculate_average(scores) highest = find_largest(scores) print(f"Scores: {scores}") print(f"Average: {avg:.1f}") print(f"Highest: {highest}") student = {"name": "Alice", "age": 20, "grade": "A"} display_student(student)

Variable Scope

Variables defined inside functions have local scope, while variables outside have global scope.

Function Best Practices
  • Use descriptive names: calculate_area() is better than calc()
  • Keep functions small: Each function should do one thing well
  • Use docstrings: Document what your function does
  • Return values: Prefer returning values over printing inside functions
Practical Example: Calculator

Let's create a simple calculator using functions to demonstrate real-world usage.

# Simple calculator functions def add(x, y): """Add two numbers and return the result.""" return x + y def subtract(x, y): """Subtract y from x and return the result.""" return x - y def multiply(x, y): """Multiply two numbers and return the result.""" return x * y def divide(x, y): """Divide x by y and return the result.""" if y == 0: return "Error: Cannot divide by zero!" return x / y # Test the calculator a, b = 10, 3 print(f"Calculator Demo with {a} and {b}:") print(f"{a} + {b} = {add(a, b)}") print(f"{a} - {b} = {subtract(a, b)}") print(f"{a} * {b} = {multiply(a, b)}") print(f"{a} / {b} = {divide(a, b):.2f}") print(f"{a} / 0 = {divide(a, 0)}")
